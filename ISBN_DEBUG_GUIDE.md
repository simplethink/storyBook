# ISBN 查询问题排查指南

## 🔍 问题现象

API 返回数据正常，但代码没有解析到数据。

**可能的原因:**
1. ❌ 域名未配置到微信公众平台白名单
2. ❌ 网络请求被微信拦截
3. ❌ HTTP 协议不支持（需要 HTTPS）

---

## 📊 查看控制台日志

重新编译小程序，扫码后查看控制台输出：

### 正常情况应该看到:
```javascript
开始查询 ISBN: 9787521758368
请求成功，statusCode: 200
ISBN API 响应：{...}
ISBN API 返回数据：{code: 0, data: {...}}
解析图书数据：大众金融
图片解析成功：["https://..."]
获取到图书信息：{title: "大众金融", ...}
```

### 异常情况:

#### 情况 1: 域名未配置
```javascript
开始查询 ISBN: 9787521758368
请求失败：{
  errMsg: "request:fail url not in domain list"
}
```

**解决方案:** 配置域名白名单（见下文）

#### 情况 2: HTTP 协议限制
```javascript
开始查询 ISBN: 9787521758368
请求失败：{
  errMsg: "request:fail http 只能访问指定的域名"
}
```

**解决方案:** 
- 配置 HTTP 白名单，或
- 升级到 HTTPS

#### 情况 3: 网络错误
```javascript
开始查询 ISBN: 9787521758368
请求失败：{
  errMsg: "request:fail timeout"
}
```

**解决方案:** 检查网络连接

---

## ✅ 解决方案

### 方案 1: 配置域名白名单（推荐）

#### 步骤:

1. **登录微信公众平台**
   - 网址：https://mp.weixin.qq.com
   - 使用管理员微信扫码登录

2. **进入开发设置**
   - 左侧菜单：开发 → 开发管理 → 开发设置

3. **配置服务器域名**
   - 找到"服务器域名"板块
   - 点击"request 合法域名"后面的"修改"按钮

4. **添加域名**
   ```
   https://data.isbn.work
   ```
   
   或者 HTTP（开发环境）:
   ```
   http://data.isbn.work
   ```

5. **保存并等待生效**
   - 点击"提交并保存"
   - 等待 5-10 分钟生效

6. **在微信开发者工具中验证**
   - 清除缓存
   - 重新编译
   - 测试扫码功能

---

### 方案 2: 使用 HTTPS（生产环境推荐）

如果支持 HTTPS，将 API URL 改为：
```javascript
const ISBN_API_BASE = 'https://data.isbn.work/openApi/getInfoByIsbn'
```

**优点:**
- 更安全
- 不受 HTTP 限制
- 生产环境必需

---

### 方案 3: 开发环境临时方案

**仅用于开发调试:**

在微信开发者工具中：
1. 点击右上角"详情"按钮
2. 勾选"不校验合法域名、web-view（业务域名）、TLS 版本以及 HTTPS 证书"
3. 重新编译

⚠️ **注意:** 此方法仅开发环境有效，真机预览和上线仍需配置域名

## 🧪 验证步骤

### 步骤 1: 检查配置
确认域名已添加到白名单：
- 微信公众平台 → 开发设置 → 服务器域名
- 查看 request 合法域名列表

### 步骤 2: 清除缓存
微信开发者工具 → 工具 → 清除缓存 → 清除全部缓存

### 步骤 3: 重新编译
点击"编译"按钮

### 步骤 4: 查看日志
打开控制台，扫码后查看完整日志

### 步骤 5: 验证结果
- ✅ 显示"获取成功" → 成功
- ❌ 提示错误 → 根据日志排查

## ⚠️ 常见问题

### Q1: 配置了域名但还是报错？
**A:** 
- 等待 5-10 分钟让配置生效
- 检查域名是否拼写正确
- 清除缓存后重试

### Q2: 开发工具有效，真机无效？
**A:** 
- 开发工具可以关闭域名校验
- 真机必须配置域名
- 按上述步骤配置域名即可

### Q3: HTTP 和 HTTPS 都要配置吗？
**A:** 
- 建议只配置 HTTPS
- 开发环境可以临时用 HTTP
- 生产环境必须用 HTTPS

### Q4: 配置后多久生效？
**A:** 
- 通常 5-10 分钟
- 最长可能需要 30 分钟
- 可以重启微信开发者工具加速生效

## 🎯 快速诊断流程

```
扫码查询
  ↓
查看控制台
  ↓
有"请求失败"日志？
  ├─ 是 → 域名未配置 → 去微信公众平台配置
  └─ 否 → 继续下一步
     ↓
有"HTTP 状态码异常"？
  ├─ 是 → API 服务问题 → 联系 API 提供方
  └─ 否 → 继续下一步
     ↓
有"API 返回错误 code"？
  ├─ 是 → ISBN 不存在或 API 限制 → 更换 ISBN 或 AppKey
  └─ 否 → 继续下一步
     ↓
有"获取到图书信息"？
  ├─ 是 → 成功！🎉
  └─ 否 → 检查页面 setData 逻辑
```
