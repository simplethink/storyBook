# 故事书小程序 📚

一个功能丰富的微信小程序，用于记录和管理你的阅读时光。

## ✨ 主要功能

### 📅 阅读日历
- 可视化日历视图，展示每天的阅读情况
- 快速查看任意日期的读书记录
- 复习提醒标记（橙色圆点）
- 阅读记录标记（绿色圆点）

### 📖 读书记录
- **扫码添加**：扫描图书条形码自动获取图书信息
- **拍照添加**：拍摄图书封面快速记录
- **手动输入**：手动填写图书信息
- 记录阅读页码、时长和笔记

### ⏰ 艾宾浩斯记忆法复习提醒
- 根据艾宾浩斯遗忘曲线自动生成复习计划
- 复习周期：1 天、2 天、4 天、7 天、15 天
- 待办任务清单
- 完成打卡功能

### 📊 数据统计
- 已读书籍数量统计
- 坚持阅读天数统计
- 待复习任务提醒
- 最近阅读记录展示

## 🚀 快速开始

### 1. 安装微信开发者工具
下载并安装 [微信开发者工具](https://developers.weixin.qq.com/miniprogram/dev/devtools/download.html)

### 2. 导入项目
- 打开微信开发者工具
- 选择"导入项目"
- 选择本项目目录 `/Users/qihao/Documents/workspace/storyBook`
- 填写你的小程序 AppID

### 3. 修改配置
在 `project.config.json` 中修改：
```json
{
  "appid": "你的小程序 AppID",
  "projectname": "storybook"
}
```

### 4. 运行项目
点击微信开发者工具的"编译"按钮即可预览

## 🔧 功能说明

### 图书信息获取
项目集成了多个免费的第三方 API 来获取图书信息：

1. **豆瓣 API**（推荐，但需要 API Key）
   - 需要在代码中替换你的 API Key
   - 信息最准确，适合中文图书

2. **Google Books API**（免费，无需 Key）
   - 作为备选方案
   - 适合外文书籍

3. **开放图书馆 API**（免费）
   - 最后的备选方案

### 数据存储
- 使用微信小程序本地存储（localStorage）
- 数据结构：
  - `readingRecords`: 阅读记录
  - `reviewTasks`: 复习任务

### 艾宾浩斯记忆法
根据德国心理学家艾宾浩斯的研究，人在学习后的遗忘曲线为：
- 20 分钟后忘记 42%
- 1 天后忘记 66%
- 2 天后忘记 72%

因此设置了 5 个关键复习节点：
1. 学习后 1 天
2. 学习后 2 天
3. 学习后 4 天
4. 学习后 7 天
5. 学习后 15 天
